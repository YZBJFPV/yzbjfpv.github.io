✅ 网络要求：连接家庭宽带（有线/无线均可），能正常访问互联网

✅ 系统要求：Windows 7/8/10/11（无需重装系统，直接在现有系统上搭建）

⚠️ 注意：关闭电脑自带的防火墙（或后续允许工具通过防火墙），避免端口被拦截

安装与配置工具

① 安装Git工具：

双击下载的 git-windows.exe，按默认选项下一步到底（无需修改任何设置），安装完成后关闭。

② 配置Nginx服务器：

1. 解压 nginx-windows.zip 到电脑任意目录（推荐 D:\yzbjfpv-server，路径不要有中文）

2. 打开解压后的文件夹，双击 nginx.exe（双击后无窗口弹出，说明启动成功）

3. 测试：打开浏览器输入 http://localhost，能看到"Nginx"页面即成功

③ 同步GitHub仓库代码：

1. 打开电脑的「命令提示符」（Win+R输入cmd回车）

2. 输入以下命令（复制粘贴即可），按回车：

cd D:\yzbjfpv-server\html
git clone https://github.com/YZBJFPV/yzbjfpv.github.io .

3. 等待1-2分钟，代码同步完成（屏幕显示"done"）

④ 配置自动同步工具：

1. 解压 sync-tool.exe 到 D:\yzbjfpv-server 目录

2. 双击运行 sync-tool.exe，点击「启动自动同步」按钮

3. 勾选「开机自动启动」，以后电脑开机就会自动同步最新代码

4

网络配置（让别人能访问你的节点）

① 检测公网IP：

双击运行 ip-check.exe，工具会自动检测你的网络是否有公网IP：

✅ 有公网IP：显示"你的公网IP：xxx.xxx.xxx.xxx"，继续下一步

❌ 无公网IP：联系群管理员获取内网穿透工具（免费）

② 路由器端口映射（关键步骤）：

1. 打开路由器管理后台（通常在路由器底部标签上，地址如 192.168.1.1 或 192.168.0.1）

2. 登录路由器（账号密码默认admin/admin，若修改过请用自己的）

3. 找到「端口映射」或「虚拟服务器」功能（不同路由器名称略有不同）

4. 添加映射规则（添加2条）：

规则1：外部端口80 → 内部端口80 → 内部IP（你的电脑局域网IP）
规则2：外部端口443 → 内部端口443 → 内部IP（你的电脑局域网IP）

5. 如何查看电脑局域网IP？：Win+R输入cmd，输入 ipconfig，找到"IPv4地址"（如192.168.1.105）

③ 测试访问：

用手机4G/5G网络（不要连WiFi），打开浏览器输入 http://你的公网IP，能看到网站页面即配置成功！

常见问题

搭建过程中遇到问题？先看这里，90%的问题都能解决！

Q：我的旧电脑配置很低，能运行吗？

A：完全可以！所有工具加起来占用内存不到100MB，CPU占用几乎可以忽略，哪怕是10年前的旧电脑也能轻松运行。

Q：真的只需要5G存储空间吗？

A：是的！GitHub仓库代码大小约200MB，加上工具和缓存，总共占用不到1GB，预留5G是为了后续代码更新和缓存，完全足够。

Q：路由器找不到「端口映射」功能怎么办？

A：不同品牌路由器的功能名称可能不同，常见的名称有「虚拟服务器」「端口转发」「NAT映射」，在路由器的「高级设置」或「网络设置」里找；实在找不到，截图路由器界面发到群里，管理员帮你找。

Q：没有公网IP，能搭建节点吗？

A：可以！联系群管理员获取免费的内网穿透工具，配置简单，同样能让外部用户访问你的节点，只是速度会比公网IP稍慢一点。

Q：电脑需要一直开机吗？偶尔关机可以吗？

A：建议7x24小时开机，这样能保证节点稳定可用；如果确实需要关机（比如出门），提前在群里说一声即可，不会影响你的贡献者资格。

Q：搭建完成后，还需要做什么维护吗？

A：几乎不需要！自动同步工具会定期拉取最新代码，Nginx会自动运行，你只需要保持电脑开机、网络通畅即可，有问题管理员会通知你。

Q：首页挂名什么时候生效？周边怎么领取？

A：节点审核通过后，24小时内会添加到首页挂名；周边会在新品发布时，管理员主动联系你，确认收货地址后免费寄送（运费由社区承担）。
